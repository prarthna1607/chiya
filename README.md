# chiya

## Site overview
A simple birthday slideshow website you can share directly.

- `index.html` is ready to use.
- It now uses external placeholder photos so you don’t need to upload local files.

## Live link via GitHub Pages
1. Commit and push:
   - `git add index.html README.md`
   - `git commit -m "ready site for live sharing"`
   - `git push origin main`
2. In GitHub > Settings > Pages:
   - Source: `main` branch
   - Folder: `/ (root)`
3. Live URL: `https://prarthna1607.github.io/chiya/`

## Optional customization
To use your own sister photos later:
- Edit `index.html` slide image `src` to your image URL, e.g. `https://example.com/my-sister.jpg` or `sister1.jpg` file in the repo.

## How to share
Send the generated GitHub Pages URL to the person, and they see the webpage immediately.
